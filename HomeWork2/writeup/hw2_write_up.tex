\section*{Design Plan}
Our Design Plan
\newline
The Clock algorithm is a disk scheduling algorithm. This works with a circular linked list which we first used in Data Structures. This determines the order for read and write requests. If there arenâ€™t any request in the head direction then the algorithm will reverse and move the opposite way. To implement this we first checked if the queue was empty and added a new request if it was. Then we moved through the queue and compared values. We then would add the new current value to the tail when we found the one next in order.
